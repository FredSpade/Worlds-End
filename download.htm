Step 1: Install the Well of Souls game engine... Download
A demo world is included called Evergreen. It's safe for younger children and demonstrates the functionality of the WoS game engine. There is no need to play it first but it's the original Well of Souls world, without which, there would be no Well of Souls or World's End.

Step 2: Install the World's End game world... Download
You must extract the zip file to your 'Worlds' folder. The directory looks something like this: C:\WoS\Worlds\WorldsEnd
The all-one-word spelling "WorldsEnd" with no apostrophe or space is used for files/folders and urls.

Step 3: Play in Single Player or Online mode
Playing (or camping and chillin') with friends is fun and exploring is easier, but it is still very possible to see as much as there is if just soloing the game.
