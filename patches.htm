This page provides links to every version of the game world that was officially released. This includes the original world file of release v1.04 and all of the patches up to the current version.

Which version do I have? Which patch should I get?
To find out, in-game type: "/version"
(without the quotes!)
Alternatively:
Explore your C:\WOS\Worlds\WorldsEnd folder in your hard-drive
or wherever you installed the game.
Double click "world.ini" to find out which version you have.

What do I do with the patch I just downloaded?
Double-click the patch and point it to the following directory:
C:\WOS\Worlds\WorldsEnd (where C: is whatever drive letter your WoS is on)
Now the patch should say you can update ok.
